# Technology Stack

## Language & Version

- Go 1.25
- Module: `github.com/Ayobami6/go-mailnow`

## Dependencies

Currently no external dependencies (standard library only).

## Common Commands

### Build
```bash
go build
```

### Test
```bash
go test ./...
```

### Run Tests with Coverage
```bash
go test -cover ./...
```

### Format Code
```bash
go fmt ./...
```

### Vet Code
```bash
go vet ./...
```

### Install Module
```bash
go mod download
```

### Tidy Dependencies
```bash
go mod tidy
```
